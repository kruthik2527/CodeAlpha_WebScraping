# CodeAlpha Web Scraping Project

## Objective
This project extracts book titles and prices from the Books to Scrape website using Python.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Website Used
https://books.toscrape.com/

## Features
- Extracts book titles
- Extracts prices
- Stores data in CSV format

## Output
The data is saved as books_data.csv
